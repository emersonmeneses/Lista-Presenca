# Lista_Presen-a
Sistema web de controle de presença escolar com login do professor e painel moderno. Permite marcar presença, ausência e observações, gerenciar turmas, visualizar total de alunos e gerar relatórios (individual, diário, mensal e geral), com opção de exportar e compartilhar. Estruturado para futura integração com Java Spring Boot.
